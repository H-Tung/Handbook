# 5.3.安装 Xorg  

在 FreeBSD 上，Xorg 可以作为包或端口安装。

二进制包可以快速安装，但自定义选项更少：

````
# pkg install xorg
````

从 Ports Collection 构建和安装：

````
# cd /usr/ports/x11/xorg
# make install clean
````

这些安装中的任何一个都会导致安装完整的 Xorg 系统。二进制包是大多数用户的最佳选择。

**x11/xorg-minimal** 中提供适合有经验的用户的较小版本的 X 系统。大多数文档、库和应用程序都不会安装。某些应用程序需要这些附加组件才能运行。
