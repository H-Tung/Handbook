# 5.3.安装 Xorg  

在 FreeBSD 上，可以作为包或 ports 安装 Xorg。

二进制包可以实现快速安装，但自定义选项更少：

````
# pkg install xorg
````

从 ports 构建和安装：

````
# cd /usr/ports/x11/xorg
# make install clean
````

通过以上任一安装方式，都会安装完整的 Xorg 系统。二进制包是大多数用户的最佳选择。

**x11/xorg-minimal** 中提供适合有经验的用户的较小版本的 X 系统。大多数文档、库和应用程序都不会安装。某些应用程序需要这些附加组件才能运行。
