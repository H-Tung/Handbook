# A.1. HTTP 镜像站

FreeBSD 的官方源代码可以通过 HTTP 从全球范围内的镜像站点获得。网站 http://ftp.FreeBSD.org/pub/FreeBSD/ 可以通过 HTTP 和匿名 FTP 获得。它由许多机器组成，由项目集群管理员操作，并在GeoDNS 后面引导用户到最近的可用镜像。

## 主镜像站

如有问题，请联系该域名的主机管理员 <mirror-admin@FreeBSD.org>。
